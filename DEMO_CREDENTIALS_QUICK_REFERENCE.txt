## 🚀 Quick Start

### Method 1: Quick Demo Login (Fastest)
1. Go to `http://localhost:5173/login`
2. See the demo credentials panel on the right
3. Click any account card
4. Form auto-fills with credentials
5. Auto-submits (800ms loading)
6. Auto-redirects to dashboard
⏱️ **Time: 5 seconds**

### Method 2: Manual Login
1. Go to `http://localhost:5173/login`
2. Copy email from above
3. Paste into email field
4. Copy password from above
5. Paste into password field
6. Click Login button
7. Wait for redirect
⏱️ **Time: 10 seconds**

### Method 3: Create New Account
1. Go to `http://localhost:5173/register`
2. Fill registration form
3. Click Register button
4. New account created
5. Auto-login and redirect
⏱️ **Time: 15 seconds**

## 🔄 How to Switch Roles

1. **Current Dashboard:** Any dashboard
2. **Click Logout:** Top right corner
3. **Redirects to:** Login page
4. **Click Demo Account:** Choose different role
5. **Auto-redirect:** New dashboard appears

**Example:**
```
User Dashboard → Logout → LoginPage 
→ Click "pro@demo.com" card 
→ Professional Dashboard
```

## ⚡ Pro Tips

### 💡 Tip 1: Save in Browser
Most browsers remember email fields
- First login: Full manual entry needed
- Subsequent logins: Browser auto-complete available

### 💡 Tip 2: Incognito/Private Mode
- No saved credentials
- Fresh session each time
- Good for testing multiple users

### 💡 Tip 3: Open Multiple Tabs
- Tab 1: Admin account
- Tab 2: Professional account
- Tab 3: User account
- Tab 4: Support account
- Switch instantly without logging out

### 💡 Tip 4: Browser DevTools
- Press F12 to open
- Go to Application/Storage
- See localStorage demo users
- Inspect form validations

### 💡 Tip 5: Create Custom Account
- Go to `/register`
- Use any name and email
- Password: minimum 6 chars
- Default role: User
- Perfect for testing registration flow

## 🐛 Troubleshooting

### ❌ Login Not Working
**Problem:** "Invalid credentials"
- ✅ Check email spelling
- ✅ Check password exactly
- ✅ Clear browser cache
- ✅ Try incognito mode
- ✅ Refresh page

### ❌ Demo Panel Not Showing
**Problem:** Demo cards not visible on login page
- ✅ Scroll right on mobile
- ✅ Check browser width
- ✅ Expand collapsible panel
- ✅ Refresh page

### ❌ Stuck on Loading
**Problem:** Spinning animation doesn't stop
- ✅ Wait 8-10 seconds (simulated delay)
- ✅ Check browser console for errors
- ✅ Try different account
- ✅ Clear localStorage

### ❌ Wrong Dashboard Showing
**Problem:** After login, showing wrong dashboard
- ✅ Check your role (email first part)
- ✅ Logout and login again
- ✅ Check URL matches expected route
- ✅ Verify credentials are correct

### ❌ Can't Register
**Problem:** Registration form won't submit
- ✅ Name must be 3+ characters
- ✅ Email must be valid format
- ✅ Password must be 6+ characters
- ✅ Passwords must match
- ✅ Email not already registered
